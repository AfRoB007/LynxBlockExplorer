API Documentation

The block explorer provides an API allowing users and/or applications to retrieve information from the network without the need for a local wallet.

API Calls

Return data from lynxd

  • getdifficulty

    Returns the current difficulty.

    {{baseUrl}}/api/getdifficulty

  • getconnectioncount

    Returns the number of connections the block explorer has to other nodes.

    {{baseUrl}}/api/getconnectioncount

  • getblockcount

    Returns the current block index.

    {{baseUrl}}/api/getblockcount

  • getblockhash [index]

    Returns the hash of the block at ; index 0 is the genesis block.

    {{baseUrl}}/api/getblockhash?index={{block.blockindex}}

  • getblock [hash]

    Returns information about the block with the given hash.

    {{baseUrl}}/api/getblock?hash={{block.blockhash}}

  • getrawtransaction [txid] [decrypt]

    Returns raw transaction representation for given transaction id. decrypt can be set to 0(false) or 1(true).

    {{baseUrl}}/api/getrawtransaction?txid={{block.txid}}&decrypt=0

    {{baseUrl}}/api/getrawtransaction?txid={{block.txid}}&decrypt=1

  • getnetworkhashps

    Returns the current network hashrate. (hash/s)

    {{baseUrl}}/api/getnetworkhashps

  • getmempoolinfo

    Returns the memory pool info

    {{baseUrl}}/api/getmempoolinfo

Extended API

Return data from local indexes

  • money-supply

    Returns current money supply

    {{baseUrl}}/ext/money-supply

  • distribution (/ext/distribution)

    Returns wealth distribution stats

    {{baseUrl}}/ext/distribution

  • address (/ext/address/hash)

    Returns information for given address

    {{baseUrl}}/ext/address/{{block.vout.0.addresses}}

  • balance (/ext/balance/hash)

    Returns current balance of given address

    {{baseUrl}}/ext/balance/{{block.vout.0.addresses}}

  • latest-txs (/ext/latest-txs?pageIndex=1&pageSize=10)

    Note: returned values are in satoshis

    {{baseUrl}}/ext/latest-txs?pageIndex=1&pageSize=10

Linking (GET)

Linking to the block explorer

  • transaction (/tx/txid)

    {{baseUrl}}/tx/{{block.txid}}

  • block (/block/hash)

    {{baseUrl}}/block/{{block.blockhash}}

  • address (/address/hash)

    {{baseUrl}}/address/{{block.vout.0.addresses}}

  • qrcode (/qr/hash)

    {{baseUrl}}/qr/{{block.vout.0.addresses}}
