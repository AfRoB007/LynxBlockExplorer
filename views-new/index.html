1 How do I move my KITTEHCOIN into my LYNX wallet?

- Download & install the Lynx wallet.

- Quit the Kittehcoin wallet. Quit the Lynx wallet. Make sure they aren’t running before you complete the next step.

- Copy the old Kittehcoin wallet.dat file. (For additional file path info see the question “How do I make my wallet sync faster?”)

- For safety, make a backup of the wallet.dat file currently in your Lynx wallet folder.

- Paste the OLD Kittehcoin wallet.dat file in to the new LYNX wallet folder, replacing the one that already exists.

- Start your Lynx wallet and let it sync.

2 What is LYNX?

Lynx is a secure cryptocurrency with fast transaction speeds and low-transaction fees. It’s also eco-friendly & easy-to-use. In 2017, our team upgraded and rebranded an existing coin (MEOW) as Lynx. How? We successfully forked Litecoin and ported the Kittehcoin blockchain history to it. This gives Lynx all the benefits of Litecoin with the full transaction history of Kittehcoin. These upgrades fixed the flaws in the original coin and revived its community of users. It’s cryptocurrency from the past; redesigned for the future.

3 Is LYNX an ICO?
